# Aryaman Awasthi - Portfolio

A dark, minimal, techy personal portfolio built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Quick start

```bash
npm install
npm run dev
```

Then open <http://localhost:3000>.

> The first run downloads the Google Fonts (Space Grotesk, Inter, JetBrains Mono)
> via `next/font`, so you need an internet connection on the first build.

## Production build

```bash
npm run build
npm run start
```
